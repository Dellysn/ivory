# IVORY

IVORY is simple and highly customizable CSS framework, gives you all to build your pages faster and easier.

## How to use

Install using npm:

```shell
npm i ivoryui-ivory
```

## Features

- HTML Reset
- Font
- Typography
- Grid
- Alerts
- Buttons
- Checkbox
- Radio
